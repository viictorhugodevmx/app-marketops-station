# 📊 MarketOps Station

Plataforma fullstack orientada a mercados financieros, construida como laboratorio profesional para simular un entorno fintech real.

## 🧠 Objetivo

Reforzar conceptos de:

- Angular (arquitectura, RxJS, estado, auth, performance)
- Node + TypeScript (API REST, WebSockets, auth)
- MongoDB (modelado básico)
- Tiempo real (market data simulado)

## 🧱 Arquitectura

marketops-station/
│
├── back/   → API (Node + TS + Mongo)
├── front/  → Angular App
│
└── README.md

## 🔌 Stack

### Backend
- Node v20.19.4
- TypeScript
- MongoDB v6.0.20
- WebSockets

### Frontend
- Angular 19.2.4
- RxJS
- Signals
